# 高考冲刺跑酷 · 张雪峰马拉松

一款类似 Chrome 离线小恐龙的网页跑酷小游戏。张雪峰在跑步机上冲刺，**跳跃**躲开地面的雪碧、**滑铲**躲开高空的雪糕，沿途**接住书本**加分。纯前端、零依赖，电脑/手机自适应。

## 玩法
- **电脑**：`空格` / `↑` 跳跃，`↓` 滑铲，`回车` 开始/重开。
- **手机**：点击屏幕跳跃，按住屏幕下方（或下滑）滑铲。
- 撞到 **雪碧 / 雪糕** 游戏结束；接住 **练习册 +1**、**志愿书 +2**；坚持越久分越高，速度会越来越快。最高分本地保存。

## 本地运行
因为用到图片资源，需要通过本地服务器打开（不能直接双击 html）：
```bash
# 在本目录下任选其一
python -m http.server 5179
# 然后浏览器访问 http://localhost:5179
```

## 部署（任意静态托管）
直接把整个文件夹上传即可，无需后端：
- **GitHub Pages**：推到仓库，Settings → Pages 选择分支根目录。
- **Vercel / Netlify**：拖拽文件夹或连接仓库，框架选 "Other / 静态"。

## 文件结构
```
index.html          页面结构 + 开始/结束界面
style.css           样式（HUD、面板、响应式）
game.js             游戏引擎（渲染、物理、碰撞、计分、输入）
assets/             素材：人物6帧、滑铲、受伤、障碍、道具、Logo、整合背景 bg.png
extract_assets.py   素材提取脚本（从『补充素材.png』『滑铲.png』『背景.png』生成 assets/）
```

## 重新生成素材
如需调整切图，编辑 `extract_assets.py` 中各区域坐标后运行：
```bash
python extract_assets.py
```
依赖：`pip install pillow numpy`
